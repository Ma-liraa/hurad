# 🚀 Hurad Dev Portfolio

A modern personal developer portfolio built with Next.js, TypeScript, and TailwindCSS to showcase frontend skills and featured projects with smooth animations and clean UI.

## ✨ Features

- ⚡ Built with Next.js (App Router)
- 🔷 Fully typed with TypeScript
- 🎨 Modern dark UI with TailwindCSS v4
- 🧩 Component-based architecture
- 🪟 Project detail modal (no route change)
- 🖼 Image slider for project previews
- 🎞 Smooth animations with Motion & motion-primitives
- 🎯 Animated icons with lucide-react
- 🧱 Accessible UI primitives via Radix UI
- 🏷 Tech stack badges per project
- 🖱 Subtle hover & shadow effects
- 📱 Fully responsive design
- 🔍 SEO meta tags included
- 📦 Static project data (no backend)
- 🚀 Ready for Vercel deployment

## 🛠 Tech Stack

- Next.js
- React 19
- TypeScript
- TailwindCSS v4
- shadcn/ui
- Radix UI
- motion / motion-primitives
- lucide-react
- clsx / tailwind-merge / CVA
